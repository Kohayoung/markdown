# 기본 명령어

CLI 명령어를 배워봅시다





$ pwd 현재 작업 폴더 출력

ctrl + l 명령어 지우기

$cd 내가 지금 활동하는 폴더를 변경

/ 루트 폴더

~ [pwd]명령어 후, 이 폴더의 축약형 단어  (home 폴더)



$ls 리스트 / 내가 지금 위치해 있는 폴더의 폴더와 파일들을 출력

ls -a 숨긴 파일도 모두 보두 보여줘

.은 현재 폴더

.. 현재 폴더 보다 상위 폴더

. 들은 숨김 폴더를 의미



$ git init

$ git add 커밋할 목록 폴더 추가

$ git commit

$git config --global user.name "Ko Hayoung"
